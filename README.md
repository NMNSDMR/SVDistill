# Сингулярное разложение (SVD) и Степенной метод

Математический вывод перехода от матричной формы SVD к векторному равенству, а также алгоритм поиска правых сингулярных векторов $v_i$ с помощью **степенного метода (Power Iteration)**.

---

## 1. Вывод векторного равенства $A v_i = \sigma_i u_i$

Мы знаем, что сингулярное разложение (SVD) матрицы $A$ записывается в матричной форме как:

$$A = U \Sigma V^T$$

Где:
* $U$ — ортогональная матрица, столбцами которой являются левые сингулярные векторы $u_i$.
* $V$ — ортогональная матрица, столбцами которой являются правые сингулярные векторы $v_i$.
* $\Sigma$ — диагональная матрица, состоящая из сингулярных значений $\sigma_i$.

---

### Шаг 1. Домножение на матрицу $V$

Так как матрица $V$ является ортогональной, для неё справедливо свойство:

$$V^T = V^{-1} \quad \Longrightarrow \quad V V^T = I$$

Домножим исходное матричное равенство **справа** на матрицу $V$:

$$A V = U \Sigma V^T V$$

Поскольку $V^T V = I$, равенство упрощается до:

$$A V = U \Sigma$$

---

### Шаг 2. Переход к покомпонентному (столбцовому) виду

Рассмотрим это умножение поэлементно по столбцам:

* **Слева:** матрица $A$ последовательно умножается на каждый $i$-й столбец матрицы $V$ (то есть на вектор $v_i$). На выходе получаем столбцы вида:
    
    $$A v_i$$

* **Справа:** умножение матрицы $U$ на диагональную матрицу $\Sigma$. Каждый $i$-й столбец результирующей матрицы получается умножением столбца $u_i$ на соответствующий диагональный элемент $\sigma_i$:
    
    $$\sigma_i u_i$$

Приравнивая левую и правую части для каждого столбца $i$, получаем фундаментальное соотношение:

> $$\bbox[10px,border:2px solid #50fa7b]{A v_i = \sigma_i u_i}$$

---

## 2. Численный поиск векторов $v_i$ и $u_i$

«На бумаге» для поиска координат векторов мы обычно решаем характеристическое уравнение для матрицы $A^T A$, находим собственные значения $\lambda_i$, а затем ищем собственные векторы. 

В программной реализации мы можем применить **степенной метод (Power Iteration)** для нахождения доминирующего вектора без прямого решения сложных уравнений.

---

### Суть степенного метода

1.  Мы берём случайный начальный вектор $v_0$ (например, заполненный единицами) и нормируем его.
2.  Любой случайный вектор $v_0$ можно представить в виде линейной комбинации собственных векторов $x_1, x_2, \dots, x_n$ матрицы $A^T A$ (которые мы пока не знаем):

    $$v_0 = c_1 x_1 + c_2 x_2 + \dots + c_n x_n$$

    *Здесь $x_1$ — собственный вектор, соответствующий наибольшему собственному значению $\lambda_1$ (наш будущий вектор $v_1$).*

3.  При однократном умножении вектора на матрицу $A^T A$ получаем:

    $$(A^T A) v_0 = c_1 \lambda_1 x_1 + c_2 \lambda_2 x_2 + \dots + c_n \lambda_n x_n$$

4.  Если мы повторим эту операцию $k$ раз (где $k$ достаточно велико, например, $k \approx 100$):

    $$\left(A^T A\right)^k v_0 = c_1 \lambda_1^k x_1 + c_2 \lambda_2^k x_2 + \dots + c_n \lambda_n^k x_n$$

---

### Геометрическая интерпретация и физический смысл

Так как $\lambda_1$ — это **наибольшее** собственное значение ($|\lambda_1| > |\lambda_2| \ge \dots$), то при возведении в большую степень $k$ коэффициент $\lambda_1^k$ начнет колоссально доминировать над всеми остальными коэффициентами $\lambda_i^k$. 

Все остальные слагаемые на фоне первого станут пренебрежимо малыми:

$$\left(A^T A\right)^k v_0 \approx c_1 \lambda_1^k x_1$$

> 💡 **Геометрически:** ортогональные преобразования выполняют повороты и растяжения пространства. Наша цель — развернуть случайный единичный вектор так, чтобы он совпал с направлением главного собственного вектора. 
>
> С каждым шагом умножения и последующего нормирования наш вектор $v$ физически **разворачивается ровно вдоль оси вектора $x_1$**. После серии итераций мы просто считываем его итоговые координаты из памяти — это и есть наш искомый вектор $v_1$.
